# kanban

Markdown kanban boards for [Claude Code](https://claude.com/claude-code). A directory of
`*.card.md` files **is** the board — no database, no daemon, plain git-friendly Markdown —
with four surfaces over it:

- **kanban** — AI-driven card management (file contracts + helper scripts)
- **web** — live browser editor (localhost server + SPA: board, dependency map, calendar, gantt)
- **cli** — conversational editor, works under remote control on mobile
- **viewer** — generated single-file HTML board for phone/tablet, read-only + queued changes

Full README, demo board, and screenshots are on their way (v1.0.0 checklist).
