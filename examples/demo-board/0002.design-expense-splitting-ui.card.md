# Design expense-splitting UI

When a group adds a shared expense, I want to see how it splits per person before confirming, so I can catch mistakes early.

## Acceptance Criteria
- Split preview updates live as amounts/participants change.
- Supports equal, percentage, and exact-amount splits.
- Works at phone width first (mobile is the primary surface).
