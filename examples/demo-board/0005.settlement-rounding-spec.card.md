# Settlement rounding spec

Decide how Tallybird rounds fractional cents when a shared expense doesn't divide evenly across participants, and who "eats" the remainder.

## Acceptance Criteria
- Rounding rule documented (largest-remainder method, remainder assigned round-robin).
- Rule covers 2-way, 3-way, and uneven-percentage splits.

## Narrative
- 2026-06-18: Settled on largest-remainder with round-robin assignment — matches how most roommates already do it by hand, so no surprises in the UI.
